给一个git分支改名的方法很简单

<!--more-->



<div id="toc"></div>

如果对于分支不是当前分支，可以使用下面代码：

```csharp
    git branch -m 原名 新
```

如果是当前，那么可以使用加上新名字


```csharp
        git branch -m 原名 
```
